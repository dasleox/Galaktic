Galaktic is a 2D Game Engine specialized in making pixel art games with ease and runable with very limited resources.

## How To Build

In your own project, clone Galaktic with this code

```
git clone https://github.com/dasleox/Galaktic
```
Build Galaktic binaries, two files will be built a .dll/.so file (Dynamic linking and static linking)
```
mkdir build && cd build
cmake .. && --build
```
